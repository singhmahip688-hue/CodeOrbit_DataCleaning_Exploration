# CodeOrbit Data Cleaning & Exploration

## Objective

This project demonstrates how to clean and analyze data using the Pandas library.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib

## Operations Performed

- Loaded CSV dataset
- Checked dataset information
- Removed duplicate records
- Converted incorrect data types
- Filled missing values
- Calculated mean and median
- Displayed value counts
- Created charts
- Saved cleaned dataset

## Output

- Clean Employee Dataset
- Statistical Summary
- Department Bar Chart
- Salary Histogram
