# CodeOrbit_DataCleaning_Exploration
# Data Cleaning Report

## Dataset

Employee Dataset

## Cleaning Steps

1. Loaded the CSV file using Pandas.
2. Checked dataset information.
3. Identified missing values.
4. Removed duplicate records.
5. Converted Age, Salary, and Experience columns to numeric data types.
6. Replaced missing Age values with the median.
7. Replaced missing Salary values with the mean.
8. Replaced missing Experience values with the median.
9. Filled missing Department values with "Unknown".
10. Generated descriptive statistics.

## Summary

The dataset was successfully cleaned by handling missing values, removing duplicates, correcting data types, and preparing it for further analysis.
